# Документация главной страницы проекта map2

Этот документ содержит полное описание структуры, компонентов, стилей и контента главной страницы проекта.

## 1. Структура страницы (`src/app/page.tsx`)

Главная страница состоит из 11 последовательных секций, импортируемых как React-компоненты. Порядок секций строго определен в `src/app/page.tsx`.

1.  **HeroSection**: Первый экран, представляющий компанию и ключевые предложения.
2.  **ServicesSection**: Секция с описанием основных услуг.
3.  **HomeAdvantagesSection**: Описание преимуществ работы с компанией.
4.  **CasesSection**: Секция с кейсами (предположительно, т.к. детально не анализировалась).
5.  **HomeReviewsSection**: Отзывы клиентов.
6.  **HomeIndustriesSection**: Отрасли, с которыми работает компания.
7.  **HomeTechnologiesSection**: Технологии, используемые в работе.
8.  **HomeAboutSection**: Краткая информация о компании.
9.  **HomeBlogSection**: Ссылки на последние статьи в блоге.
10. **HomeFaqSection**: Часто задаваемые вопросы.
11. **FinalCtaSection**: Финальный призыв к действию.

## 2. Анализ компонентов и контента

### 2.1. `HeroSection`

-   **Файлы**: `src/components/sections/HeroSection.tsx`, `HeroSection.module.css`
-   **Назначение**: Представляет компанию, содержит заголовок, подзаголовок и кнопки призыва к действию.
-   **Рендерит**:
    -   Заголовок `h1` с акцентным текстом.
    -   Параграф `<p>` с описанием.
    -   Две кнопки: "Получить консультацию" (`OpenConsultationButton`) и "Смотреть кейсы" (`Link`).
    -   Блок с тремя принципами работы, который по-разному отображается на десктопе и мобильных устройствах.
-   **Данные**: Все данные захардкожены в JSX.
-   **Пропсы**: Не принимает.
-   **Стили**:
    -   Использует CSS Modules.
    -   Фоновое изображение с эффектом "shimmer".
    -   Анимации появления элементов (`fadeUp`) и "плавания" логотипа (`heroFloat`).
    -   Сложные hover-эффекты на кнопках с использованием псевдоэлементов и transform-анимаций.

### 2.2. `ServicesSection`

-   **Файлы**: `src/components/sections/ServicesSection.tsx`, `ServicesSection.module.css`, `src/content/services.ts`.
-   **Назначение**: Демонстрирует 8 ключевых направлений услуг в виде карточек.
-   **Рендерит**:
    -   Интро-блок с заголовком, описанием, CTA-кнопкой и блоком с фактами.
    -   Сетку из 8 карточек услуг.
-   **Данные**: Импортирует `serviceGroups` из `src/content/services.ts`. Каждая карточка отображает `eyebrow`, `title`, `description`, `tags` (чипсы), `price` и `cta`.
-   **Пропсы**: Не принимает.
-   **Стили**:
    -   На мобильных устройствах сетка превращается в горизонтальный скролл.
    -   Первая карточка имеет акцентный дизайн.
    -   Карточки имеют фиксированную высоту для разных зон контента, что обеспечивает их одинаковый размер в ряду.

### 2.3. `HomeAdvantagesSection`

-   **Файлы**: `src/components/sections/HomeAdvantagesSection.tsx`, `HomeAdvantagesSection.module.css`, `src/content/home/advantages.ts`.
-   **Назначение**: Показывает преимущества работы с агентством.
-   **Рендерит**:
    -   Заголовок секции.
    -   Двухколоночный-макет: слева "липкая" (sticky) карточка с изображением, справа — контент.
    -   Контентная часть состоит из главной акцентной карточки и сетки из 4-х карточек с преимуществами.
-   **Данные**: Импортирует `homeAdvantages` из `src/content/home/advantages.ts`.
-   **Пропсы**: Не принимает.
-   **Стили**:
    -   Использует `position: sticky` для левой карточки с изображением.
    -   На мобильных макет перестраивается в одну колонку, карточка с изображением скрывается.

### 2.4. `HomeIndustriesSection`

-   **Файлы**: `src/components/sections/HomeIndustriesSection.tsx`, `HomeIndustriesSection.module.css`, `src/content/home/industries.ts`.
-   **Назначение**: Перечисляет отрасли, в которых у компании есть экспертиза.
-   **Рендерит**:
    -   Сложный двухколоночный header-grid.
    -   Сетку из 3-х карточек с отраслями.
    -   Карточку с дополнительными нишами.
-   **Данные**: Импортирует `homeIndustries` из `src/content/home/industries.ts`.
-   **Пропсы**: Не принимает.
-   **Стили**:
    -   Активно использует гриды для построения сложных макетов.
    -   На мобильных устройствах гриды перестраиваются в одну колонку.

### 2.5. `HomeTechnologiesSection`

-   **Файлы**: `src/components/sections/HomeTechnologiesSection.tsx`, `HomeTechnologiesSection.module.css`, `src/content/home/technologies.ts`.
-   **Назначение**: Демонстрирует технологический стек и подход компании.
-   **Рендерит**:
    -   Заголовок и описание.
    -   Сетку из 3-х карточек с технологиями.
-   **Данные**: Импортирует `homeTechnologies` из `src/content/home/technologies.ts`.
-   **Пропсы**: Не принимает.
-   **Стили**:
    -   "Стеклянный" дизайн карточек с фоном и границами, использующими `rgba`.
    -   На мобильных сетка превращается в горизонтальный скролл.
